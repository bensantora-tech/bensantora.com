Rust Programming

Focused on Rust across multiple domains — native desktop development using Slint for clean, performant GUIs, system-level tooling on Linux where Rust's strengths shine most naturally, and WebAssembly for bringing near-native performance to the browser.

The appeal of Rust is straightforward: memory safety without a garbage collector, a compiler that surfaces problems early, and an ownership model that rewards careful thinking about program design. These aren't theoretical benefits — they show up in production.

Current work spans desktop utilities, Linux tooling, and WASM modules. The approach is practical rather than trend-driven: build things that are correct, fast, and maintainable, and reach for Rust when those qualities matter most.
